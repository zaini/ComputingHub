What is a computer?

A computer is an electronic device which processes data into useful information. A system works to fulfill a set purpose. A computer system is a collection of hardware and software that works together to achieve a data processing task.

Data

Data is input and converted into the computer's internal representation (binary) by the input devices. The data is then processed and the returns are output by an output device. The data may be stored for later use or transmitted to another computer system.
